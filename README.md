# Vize Danışmanlık Web Sitesi

Türkiye'den dünyanın dört bir yanına vize işlemleri için bilgi sağlayan Next.js web sitesi.

## Özellikler

- **Ana Sayfa**: Ülke bayrakları ile vize bilgilerine hızlı erişim
- **Ülke Detay Sayfaları**: Her ülke için detaylı vize gereksinimleri, gerekli belgeler ve aktif konsolosluk bilgileri
- **Referanslarımız**: Şirket referansları ve başarı hikayeleri
- **Mutlu Müşteriler**: Müşteri yorumları ve memnuniyet hikayeleri
- **Bize Ulaşın**: İletişim formu ve iletişim bilgileri

## Teknolojiler

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- React

## Kurulum

1. Bağımlılıkları yükleyin:
```bash
npm install
```

2. Geliştirme sunucusunu başlatın:
```bash
npm run dev
```

3. Tarayıcınızda [http://localhost:3000](http://localhost:3000) adresini açın.

## Yapı

- `app/` - Next.js App Router sayfaları
- `components/` - Yeniden kullanılabilir React bileşenleri
- `data/` - Statik veri dosyaları (ülke bilgileri)
- `public/` - Statik dosyalar
